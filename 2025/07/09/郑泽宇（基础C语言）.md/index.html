总结

1.带有*的变量类型的标准写法：变量类型* 变量名

3.*可以是任意多个

3..带*类型的特征的赋值

4..带*类型的特征的++ –

带有*的变量的++ –

运行结果分别是101 102 104

加上/减去一个整数

总结：

1.带*类型的变量可以加，减一个整数，但不能乘或者除。

2.带*类型变量与其他整数相加或者相减时：

​ 带*类型变量 + N = 带*类型变量 + N*(去掉一个*后类型的宽度)

​ 带*类型变量 - N = 带*类型变量 - N*(去掉一个*后类型的宽度)

5..带*类型的特征求差值

总结：

1.两个类型相同的带*类型的变量可以进行减法操作。

2.相减的结果要除以去掉一个的数据宽度。

总结：

带*类型的变量，如果类型相同，可以做大小的比较。

7.带*类型的特征求值

x原来类型不带*,那么就不存在*x类型

若原来是是带*的如int *px,那么*px ==int

结论：*加变量 == 原来类型减去一个*（和&的效果刚好相反）。

解释：【ebp-8】里面存的是px的地址，把【ebp-8】里面存的px的地址取出来放到ecx中，此时ecx就为【ebp-8】内存的值（px的地址），此时【ecx】就是直接指向px,让后把【ecx】的值，也就是px地址内存的值放到edx中，假如px的值为1，那么此时edx的值就是1
